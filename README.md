# AI Customer Service & Order Management Workflow

This folder contains several workflows for handling customer service and order management tasks.  
All workflows were **created manually by a human developer** without the use of AI tools for building or coding.  

## Contents
- Workflows for receiving and responding to customer messages.
- Order management workflows: adding, editing, and canceling orders.
- Integration with Google Sheets and Telegram for record-keeping and notifications.
- Supporting nodes for data handling and file management.

> Note: While some AI nodes may exist for processing messages, the **overall workflow structure and logic are entirely human-made**.
